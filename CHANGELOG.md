# Change Log
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).


## [Unreleased]

## [0.0.16]
### Added
- Print help for specific action with action context.
- Parse options (vs only process) options.
- Adding Travis build.

### Changed
- Move to lein-git-version 1.2.7.

## [0.0.15] - 2017-01-30
### Added
- Changelog
- Accessible help function and featues (i.e. add usage etc).

### Changed
- Fixing help message printing.
- Readme documentation up to date.


## [0.0.14] - 2017-01-27
### Changed
- Custom help printing handling.
- Better help message printing


## [0.0.13] - 2017-01-20
First major release.

### Added
- Adding single command CLI functionality.
- Added easy help generation for single command CLI option.

### Changed
- Better help message printing
